About Mathelo

Mathelo was created at MHacks spring 2014 by Steven Schmatz and Amit Mizrahi, two high school students from the International Academy. It aims to be a free space for collaboration on problems of a mathematical or scientific nature.

Features

  • Real-time chat functionality
  • Freely movable and resizable windows for distraction-free working
  • Arithmetic calculations and function plotting via the Wolfram|Alpha API
  • LaTeX equation editor
  • Live video chatting
  • A whiteboard for drawing in real time
  • A minimalist, clutter-free design
