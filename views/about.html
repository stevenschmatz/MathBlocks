About MathBlocks

MathBlocks was created at MHacks spring 2014 by Steven Schmatz and Amit Mizrahi. Steven is a freshman at the University of Michigan and Amit is a senior at the International Academy in Bloomfield Hills, MI. MathBlocks aims to be a free space for collaboration on problems of a mathematical or scientific nature.

Features

  • Real-time chat functionality
  • Freely movable and resizable windows for distraction-free working
  • Arithmetic calculations and function plotting via the Wolfram|Alpha API
  • LaTeX equation editor
  • Live video chatting
  • A minimalist, clutter-free design
